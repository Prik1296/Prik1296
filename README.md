This project analyzes a real-world sales dataset to identify key business insights such as top-performing products, regions, and customers. The analysis is performed using Python and focuses on data cleaning, exploration, and visualization.

Objective :
Analyze sales performance across different countries and product lines
Identify top customers and revenue contributors
Understand monthly sales trends
Generate actionable business insights

Tools and libraries used :
Python
Pandas
Matplotlib
Jupyter Notebook

Dataset - 
Source: Kaggle (Sample Sales Data) -  https://www.kaggle.com/datasets/kyanyoga/sample-sales-data<img width="557" height="23" alt="image" src="https://github.com/user-attachments/assets/6612b7a0-514f-4628-95d9-d3a729e1e162" />
Contains information on orders, customers, products, and sales

Key Steps Performed :

1. Data Cleaning
Removed duplicate records
Handled missing values
Changed dataype format
Column renaming done.

2. Data Analysis
Total and average sales calculation
Sales by country and product line
Top customers identification
Monthly and yearly sales trend analysis

3. Data Visualization
Bar charts for sales comparison
Line chart for trend analysis
Pie chart for sales distribution

Key Insights :

Classic Cars generated the highest revenue among all product lines
USA contributed the highest share of total sales (~36%)
Sales peaked during October to December, indicating strong Q4 performance
Euro Shopping Channel was the top customer by total revenue

Conclusion :

The analysis highlights key revenue drivers and seasonal trends, helping businesses focus on high-performing products, regions, and customers to improve sales strategies.

How to Run the Project :

Download the dataset
Open the notebook in Jupyter Notebook
Run all cells to reproduce the analysis
